说明

  • 大的数据类型转换为小的数据类型时，必须强转，但有时会丢失内容
  • 数据默认值在主方法中无效，各个数据类型的默认值的使用，必须结合类才能观察到
  • 字符型数字（‘0’~‘9’）与int数字（0~9）不同，即 if(‘0’==0)结果为false.
  • 描述整数一般用int,描述小数一般用double
  • long一般用于描述日期，时间，内存或文件大小
  • byte一般用于编码转换或进行二进制流的操作
  • char一般在描述中文时用到
